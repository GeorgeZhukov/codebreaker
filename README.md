# Codebreaker

The codebreaker game written via TDD and RSpec framework.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'codebreaker', :git => "https://github.com/GeorgeZhukov/codebreaker.git"
```

And then execute:

    $ bundle


## Usage

Clone repo and run ./bin/codebreaker
